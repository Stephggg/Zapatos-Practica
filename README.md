Shoe App - Tienda de Zapatos
Descripción

Aplicación móvil desarrollada en Android utilizando Jetpack Compose. La aplicación simula una tienda de zapatos y permite al usuario navegar entre diferentes pantallas, visualizar productos y acceder a un resumen de perfil.

Objetivo del proyecto

Implementar un sistema de navegación entre pantallas en una aplicación móvil, aplicando conceptos de:

Navegación entre pantallas
Flujo de usuario
Manejo de múltiples vistas
Componentes interactivos básicos
Funcionalidades
Navegación entre tres pantallas principales:
Inicio
Productos (Detalle)
Perfil
Visualización de una lista de productos (zapatos)
Interacción mediante botones
Mensaje de confirmación al simular la compra
Flujo de navegación continuo entre pantallas

Ejecución del proyecto
Clonar el repositorio:
git clone https://github.com/tu-usuario/shoe-app.git
Abrir el proyecto en Android Studio
Ejecutar la aplicación en un emulador o dispositivo físico
Flujo de navegación
Desde la pantalla de inicio se puede acceder a productos o perfil
Desde productos se puede regresar al inicio o ir al perfil
Desde perfil se puede navegar nuevamente a productos o inicio
